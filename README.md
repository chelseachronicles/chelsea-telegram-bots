# Chelsea Updates Telegram Bot ⚽💙

A Telegram bot that automatically posts match updates for Chelsea Football Club.

## Features

- ⚽ Live score updates
- 📋 Starting lineups
- 🔥 Goal alerts
- 📊 Match statistics
- 🏁 Full-time results
- 👨 Chelsea Men's Team
- 👩 Chelsea Women's Team
- ❌ No academy matches

The bot checks live matches and sends updates automatically to a Telegram channel.

## Technologies Used

- Python
- Telegram Bot API
- Football API (API-Football / RapidAPI)
- Requests library

## Setup

1. Create a Telegram bot using BotFather.
2. Get your Bot Token.
3. Create a Telegram channel and add the bot as an admin.
4. Get your Channel Chat ID.
5. Get an API key from API-Football.

## Environment Variables

Create the following variables:

BOT_TOKEN=your_telegram_bot_token  
CHAT_ID=your_channel_chat_id  
API_KEY=your_football_api_key
